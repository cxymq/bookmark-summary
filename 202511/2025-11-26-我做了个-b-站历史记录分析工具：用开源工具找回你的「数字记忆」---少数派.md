## TL;DR
一款开源工具“B站记忆管家”，帮助用户备份、搜索和分析B站观看历史，支持数据可视化、视频下载和AI摘要，让用户完全掌控个人数据。

## Summary
1. **项目简介**：作者开发了一款名为「B站记忆管家」的开源工具，旨在帮助用户找回并分析B站观看历史记录，将历史数据转化为可搜索、可分析的个人数据资产。

2. **开发动机**：
   - **B站历史记录局限**：B站官方历史记录功能存在多个不足，包括：
     - 仅保留最近几个月记录
     - 无法进行复杂筛选
     - 缺乏数据分析功能
     - 无法导出备份数据
   - **解决方案**：通过此工具将B站历史记录变成可搜索、可分析、可掌控的数据库，彻底释放用户的数据主权。

3. **核心功能**：
   - **全面掌控历史记录**：
     - 灵活布局切换：列表/卡片视图
     - 多条件智能筛选：时间范围、UP主、视频分类、标题关键词
     - 个性化备注系统
     - 隐私保护模式：一键模糊敏感信息
     - 数据增量更新：自动比对已有数据，只获取新增记录
   
   - **数据可视化分析**：
     - 年度观看数据：热力地图和统计图表，计算全年观看总量和累计时长
     - 最爱重温榜单：统计每个视频的重看次数
     - UP主互动分析：统计关注UP主数量和平均视频完成率，双轴对比图表展示
     - 标题关键词分析：词云图捕捉高频词汇，追踪关键词月度变化趋势
   
   - **视频资源保存**：
     - 基于开源项目Yutto实现多格式、多清晰度下载
     - 可下载即将下架的视频，避免资源失效
   
   - **AI智能分析**：
     - B站AI摘要：快速回想视频内容
     - 本地AI摘要：基于Faster whisper和DeepSeek大模型，支持音频转文字和摘要生成
   
   - **自动化工作流**：
     - 计划任务系统：设置任务时间、类型、依赖项
     - 默认模板：每天凌晨自动获取新增历史记录，邮件通知运行日志
   
   - **离线图片管理**：
     - 下载封面和作者头像
     - 支持无网络环境下浏览数据
   
   - **系统设置**：
     - 数据导出功能：支持导出到Excel
     - 邮件配置：用于计划任务通知

4. **技术实现**：
   - **开源项目依赖**：
     - Yutto：视频下载功能
     - Faster whisper：音频转文字
     - DeepSeek：AI摘要生成
   - **配置要求**：
     - B站用户认证：SESSDATA
     - 邮件服务器配置
     - DeepSeek API密钥配置

5. **安装部署**：
   - **一键脚本安装**：
     ```bash
     python install_dependencies.py
     python main.py
     ```
   - **可执行文件**：
     - 提供Windows系统的exe文件
     - 分lite版（不支持音频转文字）和full版（支持所有功能）

6. **适合人群**：
   - **学习爱好者**：系统整理教学视频，构建个人知识库
   - **数据控**：分析观看习惯，量化数字生活
   - **收藏党**：永久保存珍贵视频内容，告别失效链接
